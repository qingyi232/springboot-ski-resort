================================================================================
                    IntelliJ IDEA 下载安装说明
================================================================================

⚠️ 注意：IDEA 安装包较大（约 1GB），没有包含在部署包中
请按照以下步骤自行下载安装：

================================================================================
下载步骤
================================================================================

1. 打开浏览器，复制以下网址到地址栏：
   https://www.jetbrains.com/idea/download/

2. 在页面中找到两个版本：
   - Ultimate（旗舰版，收费）
   - Community（社区版，免费）⭐ 选这个

3. 点击 Community 版本下的 "Download" 按钮
   文件名：ideaIC-2024.x.x.exe
   文件大小：约 1GB

4. 等待下载完成（根据网速，可能需要10-30分钟）


================================================================================
安装步骤（详细版）
================================================================================

1. 找到下载的 ideaIC-2024.x.x.exe 文件
   双击打开

2. 在欢迎页面，点击 "Next"

3. 选择安装路径：
   - 建议使用默认路径：C:\Program Files\JetBrains\IntelliJ IDEA Community Edition
   - 点击 "Next"

4. ⚠️ 重要：在 "Installation Options" 页面，勾选以下选项：
   [√] Create Desktop Shortcut（创建桌面快捷方式）
   [√] Update PATH variable（添加到环境变量）
   [√] Update context menu（添加右键菜单）
   [√] Create Associations（关联文件类型）
       [√] .java
       [√] .xml
       [√] .properties
   点击 "Next"

5. 点击 "Install" 开始安装

6. 等待安装完成（约 5-10 分钟）

7. 安装完成后：
   - 勾选 "Run IntelliJ IDEA Community Edition"
   - 点击 "Finish"


================================================================================
首次启动配置
================================================================================

1. IDEA 会自动启动，显示欢迎界面

2. 在 "Import Settings" 页面：
   - 选择 "Do not import settings"（不导入设置）
   - 点击 "OK"

3. 在 "Data Sharing" 页面：
   - 选择 "Don't Send"（不发送数据）

4. 在 "Customize IntelliJ IDEA" 页面：
   - 选择主题：
     * IntelliJ Light（浅色主题）
     * Darcula（深色主题）
   - 随便选一个，点击 "Next"

5. 在 "Featured plugins" 页面：
   - 可以跳过，点击 "Skip Remaining and Set Defaults"

6. 进入 IDEA 主界面，显示 "Welcome to IntelliJ IDEA"
   ✅ 安装成功！


================================================================================
验证安装
================================================================================

1. 在 IDEA 欢迎界面，点击 "New Project"

2. 如果能看到项目创建向导，说明安装成功

3. 点击 "Cancel" 取消创建

4. 关闭 IDEA，等待技术人员远程时再打开


================================================================================
配置 Maven（重要！）
================================================================================

首次使用 IDEA 打开项目时，需要配置 Maven：

1. 打开 IDEA

2. 点击菜单：File → Settings（或按 Ctrl+Alt+S）

3. 在左侧找到：Build, Execution, Deployment → Build Tools → Maven

4. 配置以下内容：
   - Maven home path: C:\Maven\apache-maven-3.x.x
   - User settings file: C:\Maven\apache-maven-3.x.x\conf\settings.xml
   - Local repository: C:\Users\你的用户名\.m2\repository

5. 点击 "Apply"，然后点击 "OK"

⚠️ 技术人员会帮你配置，这里只是说明


================================================================================
常见问题
================================================================================

Q1：下载速度很慢怎么办？
A1：可以使用迅雷等下载工具加速，或者等待技术人员远程时帮忙下载

Q2：安装时提示需要管理员权限？
A2：右键点击安装文件，选择"以管理员身份运行"

Q3：IDEA 启动很慢？
A3：首次启动会慢一些，需要初始化配置，耐心等待

Q4：IDEA 可以卸载吗？
A4：不可以！IDEA 是运行和修改项目代码必需的

Q5：Community 版本和 Ultimate 版本有什么区别？
A5：Community 版本免费，功能够用。Ultimate 版本收费，功能更多。

Q6：需要激活或注册吗？
A6：Community 版本完全免费，不需要激活或注册

================================================================================
系统要求
================================================================================

最低配置：
  - CPU：2核
  - 内存：8GB（IDEA 至少需要 2GB）
  - 硬盘：10GB 可用空间
  - 操作系统：Windows 10/11（64位）

推荐配置：
  - CPU：4核或更多
  - 内存：16GB
  - 硬盘：20GB 可用空间（SSD 更佳）
  - 操作系统：Windows 10/11（64位）
